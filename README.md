## Setup
- Create a new virtualenv named "env"
	>  python3 -m venv env
-  Activate the virtualenv
	- (OS X & Linux)
		> source env/bin/activate
	- (Windows)
		> env\Scripts\activate
- Installing dependencies
	>pip install -r requirements.txt
- Run
	> - python3 category.py 
	> - python3 Stratgery.py
	> - python3 sec_bhavadata.py
